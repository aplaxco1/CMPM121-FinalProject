CMPM 121 - Final Project
Group Members: Eduardo Alcaraz, Elizabeth Arnold, Ryan Hueckel, Autumn Plaxco


DESCRIPTION:
For this project, we decided to create a mystery scene. Within this game, you play
as a crow, searching for treasures in an old abandoned graveyard.
